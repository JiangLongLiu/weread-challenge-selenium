# 快速开始指南

本指南帮助你在 5 分钟内完成 RK3566 iStoreOS 上的微信读书助手部署。

## 环境要求

| 项目 | 要求 |
|------|------|
| 硬件 | RK3566 开发板 (如 R66S/R68S) |
| 系统 | iStoreOS |
| 网络 | 可访问互联网 |

## 部署步骤

### 第 1 步：准备 password.xls

在 `docs/rk3566-istoreos-deploy/password.xls` 中填写 SSH 密码：

| 列名 | 示例值 |
|------|--------|
| IP地址 | 192.168.123.51 |
| 用户名 | root |
| 密码 | 1234567890 |
| SSH端口 | 22 |
| 备注 | RK3566主机1 |

### 第 2 步：安装依赖

```bash
pip install pandas paramiko
```

### 第 3 步：执行部署

```bash
python docs/rk3566-istoreos-deploy/scripts/ssh_scp_util.py
```

脚本会自动：
1. 连接 SSH
2. 创建部署目录
3. 上传 docker-compose.yml
4. 删除旧容器并启动新容器

### 第 4 步：扫码登录

1. 打开 VNC 界面：http://192.168.123.51:7900
2. 输入 VNC 密码：**111111**
3. 在浏览器中打开微信读书，扫描 data/login.png 二维码登录

## 验证部署

```bash
# 查看容器状态
ssh root@192.168.123.51 "docker ps | grep weread"
```

正常输出：
```
weread-challenge-selenium-app-1      Up
weread-challenge-selenium-selenium-1   Up (healthy)
```

## 常见问题

### Q: VNC 密码不对？
A: 修改 docker-compose.yml 中的 `SE_VNC_PASSWORD` 后重新部署

### Q: 容器启动失败？
A: 检查 Selenium 是否健康：`docker ps | grep selenium`

### Q: 如何修改阅读时长？
A: 编辑 docker-compose.yml 中的 `WEREAD_DURATION=68`（分钟）

## 每日运行

部署完成后，容器会自动每日运行。你可以通过以下方式查看状态：

```bash
# 查看日志
ssh root@192.168.123.51 "docker logs weread-challenge-selenium-app-1 --tail 20"

# 查看登录二维码
ssh root@192.168.123.51 "ls -la /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/data/"
```
