# 环境配置说明

## 目标主机信息

| 项目 | 值 |
|------|-----|
| 主机名称 | RK3566主机1 |
| IP 地址 | 192.168.123.51 |
| SSH 端口 | 22 |
| 用户名 | root |
| 操作系统 | iStoreOS (基于 OpenWrt) |

## 主机环境要求

### 硬件要求
- CPU: RK3566 四核 ARM Cortex-A55
- 内存: 推荐 4GB 以上
- 存储: 需要足够空间存储 Docker 镜像和数据

### 软件要求
- Docker Engine
- Docker Compose
- 网络连接正常

## 网络配置

### 端口映射
| 容器服务 | 主机端口 | 说明 |
|----------|----------|------|
| Selenium | 4444 | WebDriver 端口 |
| VNC | 7900 | 浏览器可视化界面 |
| App | - | 无需外部端口 |

### 数据卷配置
- 部署目录: `/mnt/sata1-1/docker/mycontainers/weread-challenge-selenium`
- 数据目录: `/mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/data`
  - cookies.json - 登录凭证
  - login.png - 登录二维码
  - output.log - 运行日志
  - screenshot-*.png - 阅读截图

## 安全配置

### 凭证管理
- SSH 密码存储在 `password.csv` 文件中
- 使用 Python 脚本自动读取，避免明文密码泄露

### 防火墙规则
确保以下端口可访问：
- 22 (SSH)
- 7900 (VNC)
