# 快速开始

本指南帮助快速部署和运行 account-3016 微信读书助手。

## 环境说明

| 项目 | 值 |
|------|-----|
| 工作目录 | E:\dockers\weread-challenge-selenium-3016 |
| 部署脚本 | docs/.../account-3016/scripts/deploy.py |
| 配置文件 | docs/.../account-3016/docker-compose.yml |

## 部署步骤

### 第 1 步：执行部署

```bash
python docs/rk3566-istoreos-deploy/docker-windows-build/account-3016/scripts/deploy.py
```

### 第 2 步：扫码登录

1. 打开 VNC 界面：http://localhost:7900
2. 输入 VNC 密码：**111111**
3. 在浏览器中打开微信读书，扫描 data/login.png 二维码登录

## 常用命令

```bash
# 查看容器状态
docker ps | grep weread-challenge-selenium-3016

# 查看日志
docker logs -f weread-challenge-selenium-3016-app-1

# 停止容器
docker compose -f E:\dockers\weread-challenge-selenium-3016\docker-compose.yml down
```
