# 部署执行计划

## 任务概述
将微信读书挑战助手部署到 RK3566 iStoreOS 主机。

## 执行步骤

### 1. 创建 Git 分支
- 从 main 分支创建新分支 `rk3566-istoreos-deploy`
- 已推送到远程仓库

### 2. 创建文档目录结构
创建以下中文文档：
- 部署执行计划（本文件）
- 环境配置说明
- 部署操作手册
- 运维手册
- 故障排查指南
- Python自动化脚本说明
- README.md（文档索引）

### 3. 开发 Python 自动化脚本
- ssh_scp_util.py：从 password.csv 读取密码
- 实现 SSH 免密连接
- 实现 SCP 文件上传功能

### 4. 远程部署
- 使用 Python 脚本连接 RK3566 主机
- 上传 docker-compose.yml 到目标目录
- 在远程主机上执行容器启动

### 5. 验证部署
- 检查容器运行状态
- 验证服务是否正常
