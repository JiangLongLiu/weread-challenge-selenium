# 部署操作手册

## 准备工作

### 1. 环境检查
确保本地具备以下环境：
- Python 3.x
- pandas 库 (用于读取 Excel 密码文件)
- paramiko 库 (用于 SSH/SCP 连接)
- Git

### 2. 安装依赖
```bash
pip install pandas paramiko
```

### 3. 密码文件准备
确认 `docs/rk3566-istoreos-deploy/password.csv` 文件存在并包含正确的主机凭证。

## 自动部署步骤

### 步骤 1: 编辑 docker-compose.yml
在 `docs/rk3566-istoreos-deploy/docker-compose.yml` 中编辑配置：
- 修改端口映射、阅读时长等参数
- 配置文件修改完成后保存

### 步骤 2: 执行部署脚本
运行 Python 自动化部署脚本：
```bash
python docs/rk3566-istoreos-deploy/scripts/ssh_scp_util.py
```

该脚本将：
1. 从 password.csv 读取 SSH 密码
2. 通过 SSH 连接到 RK3566 主机
3. 创建远程部署目录
4. **上传本地 docker-compose.yml 文件** 到远程主机
5. 在远程主机上启动容器

### 步骤 2: 验证部署
部署完成后，检查容器状态：
```bash
# 在 RK3566 主机上执行
docker compose -f /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/docker-compose.yml ps
```

### 步骤 3: 获取登录二维码
1. 从远程主机下载 login.png：
```bash

```

2. 或者通过 VNC 访问：
   - 打开浏览器访问 http://192.168.123.51:7900

## 手动部署步骤（备选）

如果自动化脚本失败，可以手动执行以下步骤：

### 1. SSH 连接到主机
```bash
ssh root@192.168.123.51
```

### 2. 创建部署目录
```bash
mkdir -p /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium
cd /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium
```

### 3. 创建 docker-compose.yml
创建以下内容：
```yaml
services:
  app:
    image: jqknono/weread-challenge:latest
    pull_policy: always
    environment:
      - WEREAD_REMOTE_BROWSER=http://selenium:4444
      - WEREAD_DURATION=68
    volumes:
      - ./data:/app/data
    depends_on:
      selenium:
        condition: service_healthy
    dns:
      - 223.5.5.5

  selenium:
    image: selenium/standalone-chromium:latest
    pull_policy: if_not_present
    shm_size: 2gb
    ports:
      - "7900:7900"
    environment:
      - SE_ENABLE_TRACING=false
      - SE_BIND_HOST=false
      - SE_JAVA_DISABLE_HOSTNAME_VERIFICATION=false
    dns:
      - 223.5.5.5
    healthcheck:
      test: ["CMD", "curl", "-f", "http://selenium:4444/status"]
      interval: 5s
      timeout: 60s
      retries: 10
```

### 4. 启动容器
```bash
docker compose up -d
```

## 首次登录

1. 部署完成后，在 data 目录下找到 login.png
2. 使用微信扫描二维码登录
3. 登录成功后，cookies 会自动保存
4. 程序将自动开始阅读（默认 68 分钟）

## 配置选项

可以通过环境变量配置：

| 环境变量 | 默认值 | 说明 |
|----------|--------|------|
| WEREAD_DURATION | 68 | 阅读时长（分钟） |
| WEREAD_SPEED | slow | 阅读速度 |
| WEREAD_SELECTION | 2 | 选择书籍序号 |
| BARK_KEY | - | Bark 推送密钥 |
