# 运维手册

## 日常运维操作

### 查看容器状态
```bash
# SSH 连接到 RK3566 主机后执行
docker compose -f /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/docker-compose.yml ps
```

### 查看日志
```bash
# 查看应用日志
docker compose -f /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/docker-compose.yml logs app

# 查看 Selenium 日志
docker compose -f /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/docker-compose.yml logs selenium

# 实时查看日志
docker compose -f /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/docker-compose.yml logs -f app
```

### 重启容器
```bash
docker compose -f /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/docker-compose.yml restart
```

### 停止容器
```bash
docker compose -f /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/docker-compose.yml down
```

### 启动容器
```bash
docker compose -f /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/docker-compose.yml up -d
```

## 数据管理

### 数据目录结构
```
/mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/
├── data/
│   ├── cookies.json    # 登录凭证（重要）
│   ├── login.png       # 登录二维码
│   ├── output.log      # 运行日志
│   └── screenshot-*.png # 阅读截图
└── docker-compose.yml
```

### 备份数据
```bash
# 打包数据目录
tar -czvf weread-data-backup.tar.gz /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/data/
```

### 恢复数据
```bash
# 解压数据到原目录
tar -xzvf weread-data-backup.tar.gz -C /
```

## 定时任务

### 设置每日自动阅读
可以使用 crontab 设置定时任务：
```bash
# 编辑 crontab
crontab -e

# 添加定时任务（每天 8 点执行）
0 8 * * * docker compose -f /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/docker-compose.yml up -d
```

## 更新部署

### 更新镜像
```bash
# 拉取最新镜像
docker compose -f /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/docker-compose.yml pull

# 重启容器
docker compose -f /mnt/sata1-1/docker/mycontainers/weread-challenge-selenium/docker-compose.yml up -d
```

### 更新配置文件
1. 修改本地 docker-compose.yml
2. 使用 SCP 上传新配置
3. 重启容器
